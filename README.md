# brello board

Brello Board is a lightweight kanbard board that works using local browser storage.

**Road Map:**

- Custom labels
- Custom backrounds
- Comments on tasks
- Import JSON
- Better support for mobile

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
